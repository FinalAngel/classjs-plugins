General Help

Each example project has a SASS (Compass) file and the compiled css. Navigate to the examples/cl.{plugin} folder and compile using compass watch .

The documentation is generated using sphinx. Navigate to the docs folder and compile using make html

To auto-commit to the gh-pages branch using git push, excecute the following two lines in the terminal inside the repository:
git config --add remote.origin.push +refs/heads/master:refs/heads/gh-pages
git config --add remote.origin.push +refs/heads/master:refs/heads/master
