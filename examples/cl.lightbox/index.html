Methods

» my.lightbox.open(, [url]);
additional methods are:
close, resize, destroy,
next, previous, getElement, getCollection

Events and Callbacks

» $(document).on('cl-lightbox-open', function...);
» my.lightbox.callbacks.open = function...

available events and callbacks are:
open, close, resize, destroy, next, previous,
load, complete, unload

check the console for debug information on events.

Other

» error handling set in options.lang.errorMessage
» data attributes data-width and data-height
» url attributes ?width=1024&height=600
