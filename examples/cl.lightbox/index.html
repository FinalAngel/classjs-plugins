class.js lightbox

Examples

Released under the BSD License. The latest source can be found on Github.
Check out the Documentation or Changelog for detailed information.

This plugin has been tested in Firefox, Chrome, Safari and IE7+ and requires class.js 1.0+ and jQuery 1.4+. It is also tested under iOS, Android and Windows Phone to insure correct responsive behaviour. The lightbox is fully accessible and supports the WCAG 2.0 standards.

Types

image image group image group image group youtube vimeoh flash inline

Options

» prefix: 'myLightbox'
» group: false
» cycle: false
» modal: false
» modalClickable: false
» modalClosable: false
» forceLoad: false
» easing: 'easeInOutCubic'
» duration: 1000
» speed: 0

» fixed: false
» responsive: false
» ajax: true
» controls: false
» styles: { 'background': 'red' }
» dimensions: {}
» keys: false
» keyCodes: {}
» lang: {}

Methods

» my.lightbox.open()
available methods are:
open, close, resize, destroy,
next, previous, getElement, getCollection

Events and Callbacks

» $(document).bind('cl-lightbox-open', function...);
» my.lightbox.callbacks.open = function...
available events and callbacks are:
open, close, resize, destroy, next, previous,
load, complete, unload
check the console for debug information on events.

Other

» error handling set in options.lang.errorMessage
» data attributes data-width and data-height
» url attributes ?width=1024&height=600
